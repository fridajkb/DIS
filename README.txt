pip install:
flask
db-sqlite3

Build database:
Run the command "python3 (or your python version) builddatabase.py"

Run webapp
Run the command "python3 (or your python version) HigherLower_game.py" and open the link in a webbrowser.

Interaction:
You are given 2 movie titles, and a gross income for one of them. Then you press the higher or lower button according to if you think the other move made more or less.

You press the next button to get a different set of movies
